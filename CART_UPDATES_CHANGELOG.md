# Изменения в корзине PizzaMania

## Реализованные исправления

### 1. ✅ Исправлена проблема с кнопками +/- в корзине

**Проблема:** При изменении количества товара через кнопки +/-, количество не сохранялось на сервере и возвращалось к исходному значению при перезагрузке страницы.

**Решение:** Добавлен языковой префикс к URL обновления количества товара.

**Изменения в `cart.html`:**
```javascript
// Было:
fetch(`/cart/update_item/`, { ... })

// Стало:
const currentLang = window.location.pathname.split('/')[1] || 'ru';
fetch(`/${currentLang}/cart/update_item/`, { ... })
```

Теперь при нажатии на кнопки +/- количество товара сохраняется на сервере через AJAX-запрос с правильным языковым префиксом.

### 2. ✅ Добавлена функция восстановления удалённых товаров

**Функциональность:**
- При нажатии "Удалить" товар визуально помечается как удалённый (серый фон, полупрозрачность)
- Появляется зелёная кнопка "Восстановить" по центру карточки товара
- У пользователя есть **5 секунд** для восстановления товара
- Если товар восстановлен - он возвращается в корзину без перезагрузки страницы
- Если время истекло - товар удаляется с сервера с плавной анимацией

**Визуальные изменения:**
- Удалённый товар: `opacity-50` + серый overlay
- Кнопка восстановления: зелёная, по центру, с hover-эффектом
- Плавное удаление: transition 0.3s для opacity, maxHeight, padding, margin

**Поведение итоговой суммы:**
- Удалённые товары не учитываются в итоговой сумме
- При восстановлении товар сразу добавляется обратно в расчёт

**Изменения в коде:**

1. **Функция `recalcTotal()` обновлена:**
```javascript
function recalcTotal() {
    let total = 0;
    document.querySelectorAll('.cart-item').forEach(item => {
        // Пропускаем удалённые товары
        if (item.dataset.removed === 'true') return;
        
        const unit = parseFloat(item.dataset.unitPrice || 0);
        const qty = parseInt(item.querySelector('.quantity-input').value) || 0;
        total += unit * qty;
    });
    document.getElementById('total-price').textContent = formatPrice(total) + ' ₸';
}
```

2. **Новая логика удаления с восстановлением:**
```javascript
// При клике на "Удалить":
- Визуально помечается товар (opacity + overlay с кнопкой)
- Исключается из подсчёта итога (dataset.removed = 'true')
- Запускается таймер на 5 секунд
- При клике на "Восстановить":
  - Убирается overlay
  - Отменяется таймер
  - Товар возвращается в расчёт
- По истечении 5 секунд:
  - Отправляется запрос на удаление
  - Плавная анимация исчезновения
  - Удаление элемента из DOM
```

## Технические детали

### Используемые технологии:
- **AJAX (Fetch API)** для асинхронных запросов
- **JavaScript Dataset API** для хранения состояния
- **CSS Transitions** для плавных анимаций
- **Django Sessions** для серверного хранения корзины

### Timeout восстановления:
```javascript
const timeoutId = setTimeout(() => {
    // Удаление через 5 секунд
}, 5000);

// Сохраняем ID для возможности отмены
item.dataset.removeTimeout = timeoutId;

// При восстановлении - отменяем
clearTimeout(timeoutId);
```

### Проверка пустой корзины:
```javascript
if (document.querySelectorAll('.cart-item').length === 0) {
    window.location.reload(); // Показываем "Корзина пуста"
}
```
