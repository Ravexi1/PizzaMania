### 1. ✅ Исправлены кнопки +/- в корзине

**Решение:** Добавлен языковой префикс к URL `/cart/update_item/`

```javascript
// Добавлен языковой префикс

fetch(`/${currentLang}/cart/update_item/`, { ... })
```

### 2. ✅ Добавлена кнопка "Восстановить" при удалении
**Новая функциональность:**
- При удалении товар становится серым с overlay
- Появляется зелёная кнопка "Восстановить" по центру
- 5 секунд на восстановление
- Плавная анимация удаления

## 🧪 Как протестировать

### Тест 1: Кнопки +/-
1. Откройте http://localhost:8000/ru/
2. Добавьте товар в корзину
3. На странице корзины нажмите кнопку `+` несколько раз
4. **Перезагрузите страницу (F5)**
5. ✅ **Ожидаемый результат:** Количество сохранилось

### Тест 2: Восстановление товара
1. Добавьте 2-3 товара в корзину
2. Нажмите "Удалить" на одном товаре
3. ✅ **Проверьте:**
   - Товар стал серым и полупрозрачным
   - Появилась зелёная кнопка "Восстановить" по центру
   - Итоговая сумма пересчиталась (без этого товара)
4. Нажмите "Восстановить"
5. ✅ **Ожидаемый результат:**
   - Товар вернулся в нормальное состояние
   - Итоговая сумма обновилась (включая товар)

### Тест 3: Автоудаление через 5 секунд
1. Добавьте товар в корзину
2. Нажмите "Удалить"
3. **НЕ нажимайте "Восстановить"**
4. Подождите 5 секунд
5. ✅ **Ожидаемый результат:**
   - Товар плавно исчезнет
   - Если корзина пуста - покажется "Корзина пуста"

### Тест 4: Множественные удаления
1. Добавьте 3-4 товара
2. Нажмите "Удалить" на двух товарах
3. ✅ **Проверьте:**
   - Оба товара стали серыми с кнопками "Восстановить"
   - Итоговая сумма учитывает только оставшиеся товары
4. Восстановите один товар
5. Подождите 5 секунд для автоудаления второго
6. ✅ **Ожидаемый результат:**
   - Первый товар восстановлен и считается
   - Второй товар плавно удалился

## 🔧 Технические детали

### Файлы изменены:
- `webapp/templates/webapp/cart.html` - основная логика

### Ключевые функции:

1. **sendUpdate(item)** - обновление количества с языковым префиксом
2. **recalcTotal()** - пересчёт итога с учётом удалённых товаров
3. **remove-btn handler** - новая логика удаления с восстановлением

### Таймауты:
- **5 секунд** на восстановление товара
- **0.3 секунды** анимация удаления

### Dataset атрибуты:
- `data-removed="true"` - товар помечен как удалённый
- `data-remove-timeout` - ID таймера для отмены
